认识工程


主要的话也就是这一部分。

java里就是你的源代码，用来实现具体的功能，可以理解为是后端。

res里主要是各种资源文件，比如要调用的图片，按钮，组件等等，可以理解为是前端。


一些依赖需要添加在build.gradle.kts中，个人理解有点像python中的pip，先要在这里添加，然后才能在需要用到的地方import。

dependencies {
    implementation("androidx.appcompat:appcompat:1.6.1")
    implementation("androidx.core:core-ktx:1.12.0")
    implementation("androidx.recyclerview:recyclerview:1.3.0")
    implementation("com.google.code.gson:gson:2.10.1")
}

这是面向AI编程中常用的几个依赖，主要是添加AppCompatActivity类，经常被用作活动的基类。

使用时记得要在themes.xml中修改当前主题

<?xml version="1.0" encoding="utf-8"?>
<resources>

    <style name="Theme.SimpleExpenseApp" parent="Theme.AppCompat.Light.NoActionBar">
        <!-- 可先留空 -->
    </style>
</resources>

好了，你现在已经是一个成熟的安卓开发者了，快去开发属于你自己的APP吧。
