基于STM32CubeMX创建platformio工程

2025/5/29 嵌入式 stm32

生成代码时，Toolchain/IDE 选择 Makefile，如果是已经生成过的工程，就先另存为一个新的工程，再生成。

然后在PIO Home中选择New Project。

注意，这里的工程名和选择的地址都要和STM32CubeMX中对应的部分完全一致。

最后一步，编写platformio.ini文件：

[platformio]
src_dir = .
default_envs = black_f407zg ;默认环境

[env:black_f407zg]
platform = ststm32
board = black_f407zg
framework = stm32cube
build_flags = 
    -ICore/Inc              ; -I：包含的库
    -ICMSIS/Core/Include
    -D USART_SEND_DATA      ; -D：宏定义
build_src_filter = 
    +<Core/**>              ; 要编译的文件，这里是Core下的所有文件
debug_tool = jlink
upload_protocol = jlink

相关的指令还有很多，不过有这些基本的就可以编译运行了。
