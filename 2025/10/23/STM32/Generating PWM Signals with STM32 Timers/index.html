STM32定时器生成PWM波

2025/10/23 嵌入式 stm32

那么言归正传，在中断一节只讲了基本定时器，但基本定时器不具备生成PWM波的功能，这次使用通用定时器。

原理讲解

上图为向上计数模式。

在PWM输出模式下，除了CNT（计数器当前值）、ARR（自动重装载值）之外，还多了一个值CCRx（捕获/比较寄存器值）。当CNT小于CCRx时，TIMx_CHx通道输出低电平。当CNT等于或大于CCRx时，TIMx_CHx通道输出高电平。

所以PWM的一个周期为

  1. 定时器从0开始向上计数
  2. 当0-t1段,定时器计数器TIMx_CNT值小于CCRx值,输出低电平
  3. t1-t2段,定时器计数器TIMx_CNT值大于CCRx值,输出高电平
  4. 当TIMx_CNT值达到ARR时,定时器溢出,重新向上计数…循环此过程
  5. 一个PWM周期完成

所以设置的时钟的频率直接决定了PWM的频率精度、频率范围和占空比分辨率。

配置步骤

时钟配置为100kHz，PWM部分的参数解释如下

参数/配置项 含义与作用 常见配置选项/建议
Mode 设置PWM的工作模式，决定了计数值与比较值如何控制输出电平 - PWM Mode 1：向上计数时CNT < CCRx为有效电平
- PWM Mode 2：向上计数时CNT < CCRx为无效电平
Pulse 设定PWM的初始占空比，对应捕获/比较寄存器(CCRx)的值 取值范围：0 到 Counter Period(ARR)，决定初始占空比大小
Output Compare Preload 启用输出比较预加载功能 - 使能：修改CCRx值在下次更新事件生效，防止毛刺
- 禁用：立即修改CCRx值，可能导致当前周期异常
Fast Mode 快速模式，影响输出比较的响应方式 - 使能：匹配时立即置位标志，响应更快
- 禁用：在更新事件时处理标志位，通用场景
CH Polarity 设置PWM通道的输出极性 - High：有效电平为高电平
- Low：有效电平为低电平

占空比将在程序中动态修改，这里就先配置成0。

使用

HAL_TIM_PWM_Start(&htim4,TIM_CHANNEL_1); // 记得先开启

__HAL_TIM_SET_COMPARE(&htim4, TIM_CHANNEL_1, 500); // 程序中动态控制占空比
