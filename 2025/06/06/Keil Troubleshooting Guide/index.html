KEIL

微库（Microlib）

这是为了小内存嵌入式系统设计的，用于代替标准C语言库的库，同时也可以实现一些特殊的功能。

在这个地方开启：

有一个很方便的用途是，开启微库后，可以通过重写fputc()函数来直接用printf()进行串口输出。

int fputc(int ch, FILE *f)
{
 uint8_t temp[1] = {ch};
 HAL_UART_Transmit(&huart2, temp, 1, 200);
 return ch;
}

但这玩意的执行效率显著低于直接HAL_UART_Transmit()，所以慎用。

添加GD32的支持

  1. ctrl+shift+p​打开命令窗口
  2. 输入PlatformIO: Open PlatformIO Core CLI​
  3. 在打开的终端中执行下面的命令
pio platform install https://github.com/CommunityGD32Cores/platform-gd32.git

然后就可以用了
