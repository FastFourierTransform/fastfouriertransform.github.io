Cherrypy入门

2024/10/17 全栈开发 后端 Cherrypy python

借此机会顺便补一下计网的相关知识，大概不会太复杂，所以直接开始吧。

pip install cherrypy
import cherrypy

class HelloWorld:
    @cherrypy.expose
    def index(serlf):
        return "hello world!"
cherrypy.quickstart(HelloWorld())

这里涉及到一个装饰器的概念。

装饰器（Decorator）是Python中的一种高级函数，主要用于在不改变原函数代码的情况下，动态地修改或扩展函数或类的功能。它可以通过在函数或方法前添加装饰器来执行额外的逻辑，例如日志记录、权限检查、性能监测等。

装饰器本质上是一个接受函数作为参数的函数，并返回一个新的函数或增强后的版本。它通常通过**@符号**来简化语法，应用在函数或方法定义的上方。

一个最简单的装饰器形式如下：
的

def my_decorator(func):
    def wrapper():
        print("装饰器逻辑：在原函数执行之前")
        func()  # 执行原函数
        print("装饰器逻辑：在原函数执行之后")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()

输出结果为：

装饰器逻辑：在原函数执行之前
Hello!
装饰器逻辑：在原函数执行之后

@cherrypy.expose：这是一个 CherryPy 自带的装饰器，用来将某个方法暴露为可通过 HTTP 访问的 Web API 路由。

然后还有其他两个常用的装饰器：

@cherrypy.tools.json_in()
@cherrypy.tools.json_out()

用于通过json文件实现输入输出。

import requests

# 获取网页输出
requests.get(URL).json()

# 输入json文件，并获取输出
requsets.post(URL,json_file).json()
